# 可遇可求-Cansolve - BNBU 留学规划助手 (MVP)

这是一个专为北师港浸大（BNBU）学生开发的留学申请辅助工具 MVP。

本项目旨在解决 BNBU 学生在申研过程中面临的信息不对称和规划迷茫问题。通过数据可视化和集成 AI 大模型能力，为学生提供客观的选校定位和可执行的学业规划路径。

![项目预览图](/public/logo.png) 

## 🎯 核心功能模块

本项目 MVP 目前包含三个核心功能模块，构成了从"目标定位"到"路径规划"再到"实时咨询"的闭环。

### 1. 动态选校定位 (AI Matcher)

帮助学生快速建立合理的申请目标预期。

* **引导式信息采集**：通过简洁的对话流界面，收集学生的专业背景、当前 GPA 和意向留学地区。

* **GPA 动态推演**（核心功能）：提供一个交互式 GPA 滑块。学生拖动滑块调节假设的 GPA，界面右侧的院校列表会实时更新其匹配状态（冲刺/匹配/保底），直观展示每一分绩点提升带来的择校范围变化。

### 2. 学业路径规划 (Strategic Planner)

将长期的申请目标拆解为短期的学业行动指南。

* **目标差距可视化**：在页面顶部清晰展示当前累计 GPA 与设定梦校最低要求之间的精确差距。

* **期末成绩模拟器**（核心功能）：列出下学期的核心课程，允许学生逐一调整每门课的预期目标成绩（A至C）。系统会自动计算模拟后的累计 GPA。

* **达标预警**：如果模拟结果低于目标要求，系统会发出明确警报，提示学生调整选课或复习策略。

### 3. 场景化 AI 顾问 (Context-Aware Chatbot)

集成 DeepSeek API，提供比传统问答更精准的咨询服务。

* **全局悬浮入口**：在应用的任何页面均可随时唤起对话窗口。

* **上下文感知能力**：AI 不仅回答通用问题，还能感知用户当前所在的页面环境（是在进行选校还是在做规划），从而提供与当前任务高度相关的建议，无需用户重复背景信息。

## ⚙️ 技术实现概览

本项目采用现代化的全栈架构构建，确保了良好的用户体验和扩展性。

* **前端框架**: Next.js 14 (App Router) + TypeScript

* **UI 组件库**: Shadcn/ui + Tailwind CSS (采用深色模式设计，注重信息呈现的清晰度)

* **交互动画**: Framer Motion (用于平滑的状态切换和数据反馈)

* **AI 集成**: Vercel AI SDK (处理流式对话) + DeepSeek API (后端模型支持)

* **状态管理**: React Context API (实现跨组件的 AI 上下文共享)

## 🚀 本地开发指南

1.  **克隆仓库**

    ```bash
    git clone https://github.com/Boyi-Wang-washu/BNBU_Study_abroad_AI_Agent.git
    ```

2.  **安装依赖**

    ```bash
    npm install
    ```

3.  **配置环境变量**

    在项目根目录创建 `.env.local` 文件，并填入你的 API 密钥：

    ```env
    DEEPSEEK_API_KEY=your_actual_api_key_here
    ```

4.  **启动开发服务器**

    ```bash
    npm run dev -p 3020
    ```

    访问 `http://localhost:3020` 查看效果。
